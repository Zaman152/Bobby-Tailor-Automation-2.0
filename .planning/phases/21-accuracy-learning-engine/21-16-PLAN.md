<threat_model>
ASVS L1 — the learning-poisoning surface:
- **Feedback amplification (HIGH — blocking):** a wrong learned value silently poisoning geometry-derived numbers is the phase's named catastrophic failure. The guardrail is enforced at THREE seams and tested at each: aggregation only renames (never re-quantifies); calculator learned heights only when no deterministic source AND flagged; scale prior only when solver confidence < HIGH... including a direct test that a HIGH-confidence geometry quantity survives a contradicting learned value unchanged.
- **Prompt injection via learned strings (MEDIUM):** vocabulary/expected-item strings originate from human input; render_prompt_hint wraps them in the `<pdf_text>`-style data delimiters (21-08 convention) and caps hint length (2,000 chars).
- Store reads are parameterized (21-04); no new SQL here — retrieval.py uses store APIs only (grep: no sqlite3 import).
</threat_model>
