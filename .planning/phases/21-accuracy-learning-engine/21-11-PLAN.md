## Artifacts this phase produces

New symbols created by this plan:
- **Dataclass** `EnsembleResult` — `value: Optional[float]`, `votes: list`, `confidence: str`, `needs_review: bool`, `spread: float`, `n_samples: int`, `method: str` ('vote_median'|'tiled+vote'), `to_dict()`
- **Function** `vote(samples: list[float], unit_kind: str) -> EnsembleResult` — EA: median; accept if max−min ≤ max(1, 10% of median); SF/LF (raster only): median, accept if spread ≤ 10%; else needs_review. Agreement mapping: unanimous→high, majority-within-tolerance→medium, none→low+needs_review
- **Function** `run_ensemble(analyze_call, n: int = None, escalate_on_disagreement: bool = True, budget: BudgetGuard = None) -> EnsembleResult` — fans out N calls via the injectable `analyze_call`, N=3 default, +2 more when first 3 disagree; diversity via temperature 0.4-0.7 with prompt-variation fallback (tile offsets/ordering) if the model ignores temperature (RESEARCH open question 3 — probe at runtime once and cache the answer)
- **Function** `run_tiled_count(page_image_or_page, grid, analyze_call, budget) -> float` — integrates existing `COUNT_TILING` machinery (accuracy_config.count_tiling_grid), 15-20% tile overlap, position-based NMS dedup in the overlap band; result becomes ONE ensemble member
- **Class** `BudgetGuard` — `__init__(max_usd: float)`, `charge(cost_usd)`, `remaining`, raises `BudgetExceeded`; consumed by ensemble, verify-retry (21-14), and the pipeline (21-13)
- **Exception** `BudgetExceeded`
- **Config**: `API_BUDGET_PER_RUN_USD` in config.py (env, default 10.0), `ENSEMBLE_N` (env, default 3); both in .env.example and conftest pinning

<threat_model>
ASVS L1 — API cost + key surface:
- **Cost explosion (HIGH — the named V3-PROD-02 risk):** N×tiles×retries multiplies. Mitigation is structural: every API call in ensemble paths must pass through `BudgetGuard.charge()` using the `_cost_usd` stamp; on BudgetExceeded the pipeline marks remaining items needs_review with reason "budget_exhausted" — never a silent partial (Pitfall 7 policy verbatim). Blocking acceptance test.
- **API key:** calls go through the existing claude_analyzer client — no new key handling.
- Ensemble prompt content is candidate-ID-grounded (21-08 schemas) — injection surface unchanged.
</threat_model>
