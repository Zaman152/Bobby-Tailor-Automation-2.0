New symbols created by this plan:
- **Dataclass** `WallSegment` — `centerline_pt: tuple`, `length_pt: float`, `thickness_pt: float`, `angle_deg: float`, `thickness_class: str` (e.g. "TC1"), `openings_pt: list[tuple]`, `candidate_id: str` ("WALL-TC1" per class), `confidence: str`, `to_dict()`; scale-independent (points until bound — RawGeometry pattern)
- **Function** `extract_walls(page, fpp: float, exclude_rects: list = None) -> list[WallSegment]` — Recipe B (angle grouping, offset pairing at 3"-16" real thickness via fpp, ≥70% projection overlap, collinear chain merge with door-gap tolerance, Bézier flattening at 0.5pt chordal tolerance for curved walls)
- **Function** `wall_lf_by_class(walls, fpp) -> dict[str, float]` — LF sums per thickness class
- **Function** `opening_gap_widths(walls, fpp) -> list[float]` — feeds scale solver door_check
- **Dataclass** `RoomPolygon` — `polygon_wkt: str`, `area_pt2: float`, `label: str`, `label_source: str` ('text_in_polygon'|'vision'|'none')`, `candidate_id: str` ("ROOM-n"), `confidence: str`, `to_dict()`
- **Function** `extract_rooms(page, walls, viewport_bbox, fpp) -> list[RoomPolygon]` — Recipe C (node network via unary_union, polygonize, discard exterior/slivers <10 SF, bind get_text words labels by point-in-polygon)
- **Function** `building_footprint(rooms_or_walls, fpp) -> dict` — exterior union area/perimeter + `dims_crosscheck` field comparing against footprint_takeoff overall-dims result (≤2% agree, else needs_review)
- **Curve support flag**: sheets where Bézier length ≥5% of linework set `needs_review` note "curve support verification" (Pitfall 6)
