New symbols created by this plan:
- **Dataclass** `SymbolGroup` — fields: `candidate_id: str` (e.g. "SYM-3"), `count: int`, `signature_hash: str`, `instance_bboxes_pt: list[tuple]`, `representative_bbox_pt: tuple`, `bbox_size_ft: tuple`, `contained_texts: list[str]`, `confidence: str`, `needs_review: bool`, `to_dict()`
- **Function** `cluster_symbols(page, fpp: float, exclude_rects: list = None) -> list[SymbolGroup]` — Recipe D pipeline (fpp = feet-per-point from the scale solver; used for size filters)
- **Function** `_shape_signature(cluster) -> str` (internal) — translation-normalized quantized-tuple multiset hash, min over 8 dihedral transforms
- **Function** `_collapse_hatch_fields(segments)` (internal) — dense same-angle same-spacing group collapse (Pitfall 2), returns (kept_segments, hatch_regions)
- **Dataclass** `ScheduleTable` — `kind: str` ('door'|'window'|'finish'|'fixture'|'equipment'|'keynote'), `rows: list[dict]`, `page_index: int`, `bbox_pt: tuple`, `confidence: str`
- **Function** `extract_schedules(doc, kinds: list = None) -> list[ScheduleTable]` — generalized find_tables loop with per-kind token vocabularies
- **Constant** `SCHEDULE_KIND_VOCABULARIES: dict[str, dict]` in tables.py — per-kind tag/header token sets

<threat_model>
ASVS L1 — PDF parsing of untrusted files:
- **Resource exhaustion via pathological geometry (MEDIUM):** clustering is the most combinatorial deterministic step (union-find over up to 68k+ primitives). Mitigations: grid-hash spatial bucketing (never O(n²) pairwise), hard cap of 200k primitives per page for clustering (above → return [] with logged warning), hatch collapse BEFORE clustering.
- **Malformed table content (LOW):** find_tables output cells pass through `_cells`-style sanitization (strip, length-cap 500 chars/cell) before entering rows that later reach prompts/UI.
- No network, SQL, or writes.
</threat_model>
