## VPS Deployment (Ubuntu)

### System Requirements

- Ubuntu 20.04+ or Debian 11+
- Python 3.10+
- 2GB RAM minimum (4GB recommended for concurrent jobs)
- Chromium system dependencies

### Installation

```bash
# Update system
sudo apt update && sudo apt install -y python3-pip python3-venv

# Clone and setup
cd /home/ubuntu
git clone <repo-url> bobby-tailor
cd bobby-tailor
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Install Playwright browser and system dependencies
playwright install chromium
playwright install-deps chromium  # Installs libnss3, libatk1.0-0, etc.

# Configure environment
cp .env.example .env
# Edit .env with your credentials
```

### Chromium Launch Flags

The browser is configured with VPS-safe flags:

- `--no-sandbox`: Required in restricted environments
- `--disable-dev-shm-usage`: Routes shared memory to `/tmp` (prevents crashes when `/dev/shm` is limited to 64MB)
- `--disable-blink-features=AutomationControlled`: Reduces automation detection

### Running with Gunicorn

```bash
# Production server
gunicorn -w 2 -b 0.0.0.0:5050 app:app --timeout 300

# With systemd service (recommended)
sudo systemctl enable bobby-tailor
sudo systemctl start bobby-tailor
```

### Docker Deployment (Alternative)

If using Docker, configure shared memory:

```yaml
# docker-compose.yml
services:
  bobby-tailor:
    build: .
    ipc: host  # OR shm_size: 1gb
    mem_limit: 2g
```

### Troubleshooting

**Browser crashes with "Target closed":**
- Check `/dev/shm` size: `df -h /dev/shm`
- Ensure `--disable-dev-shm-usage` is in launch args
- Check system memory: `free -h`

**Screenshots are blank:**
- Verify `playwright install chromium` completed successfully
- Check logs for canvas stability timeout warnings
- Increase `CANVAS_STABILITY_TIMEOUT` in `.env`
```
