Create `job_store.py` in the project root following the same patterns as `stackct_store.py`

**Schema — add to the existing `stackct.db` via `CREATE TABLE IF NOT EXISTS job_runs`:**

```sql
CREATE TABLE IF NOT EXISTS job_runs (
    job_id              TEXT PRIMARY KEY,
    job_type            TEXT NOT NULL,          -- 'stackct' | 'pdf'
    project_name        TEXT,
    status              TEXT,                   -- raw: done | error | cancelled
    outcome             TEXT,                   -- derived: success | partial | failed | cancelled
    error_message       TEXT,
    warning_message     TEXT,
    mode                TEXT,                   -- all | specific | analyze_only
    mode_detail         TEXT,                   -- full | analyze_only
    started_at          TEXT,                   -- ISO 8601
    finished_at         TEXT,
    duration_sec        REAL,
    sheets_total        INTEGER,
    sheets_succeeded    INTEGER,
    sheets_failed       INTEGER,
    linked_sheets_added INTEGER,
    progress_final      INTEGER,
    run_folder          TEXT,
    report_json_path    TEXT,
    log_tail_json       TEXT,                   -- JSON array, capped at 80 entries
    meta_json           TEXT
);
```

**Functions to implement:**

`_derive_outcome(job: dict) -> str`
Pure function — testable in isolation.
- `status == "cancelled"` → `"cancelled"`
- `status == "error"` → `"failed"`
- `status == "done"` and (`result.get("partial")` or `job.get("warning")`) → `"partial"`
- `status == "done"` (no warnings) → `"success"`
- Fallback → `"failed"`

`_ensure_schema() -> None`
Check `_initialized` flag; if False, open connection, run `conn.executescript(SCHEMA_SQL)`,
commit, close, set flag. Use module-level `_write_lock` for thread safety.

`save_job_run(job: dict, job_id: str, job_type: str) -> None`
Extract fields from `job` dict (project, status, result subdict, log list).
- `log_tail_json`: take last 80 entries from `job.get("log", [])`, store as `json.dumps()`
- `sheets_total`: `result.get("sheets_total") or result.get("total_sheets")`
- `sheets_succeeded`: `result.get("sheets_succeeded") or result.get("sheets_processed", 0)`
- `sheets_failed`: `len(result.get("sheets_failed") or [])`
- `linked_sheets_added`: `job.get("linked_sheets_count", 0)`
- `run_folder`: `result.get("_run_folder_name")`
- `duration_sec`: compute from `started_at` and `finished_at` if both present; use `datetime.fromisoformat()` subtraction; store as `total_seconds()`; handle ValueError silently
- Use `INSERT OR REPLACE INTO job_runs` so re-running save on same job_id is idempotent
- After insert, if `JOB_HISTORY_RETENTION_DAYS > 0`, run:
  `DELETE FROM job_runs WHERE started_at < datetime('now', '-{N} days')`
  using the constant. Wrap entire write in `_write_lock`.

`list_job_runs(limit: int = 50, offset: int = 0, outcome: str | None = None) -> list[dict]`
SELECT all columns except `log_tail_json` and `meta_json` (summary only).
ORDER BY `started_at DESC`. Apply WHERE `outcome = ?` if outcome is not None.
Return list of dicts via `[dict(row) for row in cursor.fetchall()]`.

`get_job_run(job_id: str) -> dict | None`
SELECT all columns including `log_tail_json`. Parse `log_tail_json` back to list before
returning. Return None if not found.

Add module-level docstring: "SQLite persistence for job run history. Uses same stackct.db file."
Use `from __future__ import annotations` and type hints throughout.
Add `logger = logging.getLogger(__name__)` and log errors with `logger.exception()`.

**app.py — wire _finalize_stackct_job:**
`_finalize_stackct_job` has three terminal code paths. Add `job_store.save_job_run()` at the
end of each:

1. Non-dict result (early return after setting status=error):
   ```python
   job_store.save_job_run(jobs[job_id], job_id=job_id, job_type="stackct")
   return
   ```

2. Cancelled path (early return after setting current_phase="cancelled"):
   ```python
   job_store.save_job_run(jobs[job_id], job_id=job_id, job_type="stackct")
   return
   ```

3. Error path (early return after setting status=error):
   ```python
   job_store.save_job_run(jobs[job_id], job_id=job_id, job_type="stackct")
   return
   ```

4. Done path — at the very end of the function (after warning or success log):
   ```python
   job_store.save_job_run(jobs[job_id], job_id=job_id, job_type="stackct")
   ```

**app.py — wire _pdf_job:**
In `_pdf_job`, in the try block after setting `status = "done"` and progress:
```python
job_store.save_job_run(jobs[job_id], job_id=job_id, job_type="pdf")
```
In the except block after setting `status = "error"`:
```python
job_store.save_job_run(jobs[job_id], job_id=job_id, job_type="pdf")
```
Wrap each `save_job_run` call in try/except Exception so a DB failure never crashes a job:
```python
try:

except Exception:
    logger.exception("Failed to persist job run %s to history", job_id)
```
