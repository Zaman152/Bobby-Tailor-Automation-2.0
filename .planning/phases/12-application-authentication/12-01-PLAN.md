  <action>
Create `seed_admin.py` in project root with:

1. Imports: `pathlib.Path`, `flask_bcrypt.Bcrypt`, `dotenv.set_key`
2. Constants:
   - `ADMIN_EMAIL = "admin@bobbytailor.com"` (from 12-CONTEXT.md)
   - `ADMIN_PASSWORD = "BobbyTheAdmin@1"` (from 12-CONTEXT.md)
   - `ENV_PATH = Path(__file__).resolve().parent / ".env"`
3. Generate bcrypt hash with rounds=12: `_bcrypt.generate_password_hash(ADMIN_PASSWORD, rounds=12).decode("utf-8")`
4. Delete ADMIN_PASSWORD from memory immediately: `del ADMIN_PASSWORD`
5. Create .env if not exists: `ENV_PATH.touch()`
6. Write to .env using `set_key()`:
   - `ADMIN_EMAIL`
   - `ADMIN_PASSWORD_HASH`
7. Print confirmation (email only, NEVER print password or hash)

Include docstring explaining this is a one-time script for VPS setup. Add `#!/usr/bin/env python3` shebang.

CRITICAL: NEVER log, print, or return the plaintext password. Delete from memory after hashing.
  </action>
  <verify>
Run `python seed_admin.py` — outputs "Admin seeded: admin@bobbytailor.com".
Run `grep ADMIN_PASSWORD_HASH .env` — shows bcrypt hash starting with `$2b$12$`.
  </verify>
  <done>seed_admin.py exists, generates bcrypt hash for BobbyTheAdmin@1, writes ADMIN_EMAIL and ADMIN_PASSWORD_HASH to .env.</done>

```bash
# Leave unset in production (defaults to secure mode)
# FLASK_ENV=development
```
