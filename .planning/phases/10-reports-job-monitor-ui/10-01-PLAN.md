# Plan 10-01: Reports Page + Preview Tabs

## Overview

**Goal**: Build the Reports page with expandable report cards and in-browser preview tabs for Summary, Calculations, Raw Items, and JSON views.

**Master Reference**: §8.5 (Page: Reports), §7.1.2 (In-Browser Report Preview)

**Wave Dependencies**:
- Phase 5 (API endpoints): `/api/reports`, `/api/reports/<folder>/<file>`, preview endpoint
- Phase 7 (Shell & Navigation): Sidebar navigation, main content layout
- Phase 8 (Page components): Card components, button styling, page shell

## Success Criteria

1. Reports page lists all report runs as expandable cards
2. Each card shows: project name, date, sheet count, raw items count, calculated count, API cost
3. Preview button expands card to show tabbed preview panel
4. Tabs: Summary (default) | Calculations | Raw Items | JSON
5. Calculations tab shows sortable, filterable data table with search
6. Download buttons work without closing preview panel
7. Preview calls use Phase 5 API endpoints

### Task 7: JSON Tree Viewer Tab
**File**: `static/app.js`

```javascript
async function loadJsonTab(folder) {
  const res = await fetch(`/api/reports/${encodeURIComponent(folder)}/preview/takeoff.json`);
  const data = await res.json();
  
  return renderJsonTree(data.data);
}

function renderJsonTree(obj, depth = 0) {
  if (typeof obj !== 'object' || obj === null) {
    return `<span class="json-value">${JSON.stringify(obj)}</span>`;
  }
  
  const isArray = Array.isArray(obj);
  const entries = Object.entries(obj);
  
  return `
    <div class="json-node" style="margin-left: ${depth * 16}px">
      <span class="json-bracket">${isArray ? '[' : '{'}</span>
      ${entries.map(([key, val]) => `
        <div class="json-entry">
          ${!isArray ? `<span class="json-key">"${key}"</span>: ` : ''}
          ${typeof val === 'object' && val !== null 
            ? `<span class="json-toggle" onclick="toggleJsonNode(this)">▶</span>${renderJsonTree(val, depth + 1)}`
            : renderJsonTree(val, depth + 1)
          }
        </div>
      `).join('')}
      <span class="json-bracket">${isArray ? ']' : '}'}</span>
    </div>
  `;
}
```

**Acceptance**:
- JSON renders as collapsible tree
- Click toggle expands/collapses nodes
- Arrays and objects visually distinct
- Primitive values styled appropriately
