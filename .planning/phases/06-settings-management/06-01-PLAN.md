<task type="auto">
  <name>Task 1: Create settings.py module</name>
  <files>settings.py (new)</files>
  <action>
Create `settings.py` in project root with:

1. **Constants:**
   - `ENV_PATH = Path(__file__).resolve().parent / ".env"`
   - `SENSITIVE_FIELDS = {"STACKCT_PASSWORD", "ANTHROPIC_API_KEY"}`
   - `ALLOWED_FIELDS` — whitelist of all configurable settings from `.env.example`
   - `VALID_MODELS = ["claude-haiku-4-5", "claude-sonnet-4-6", "claude-opus-4-7"]`
   - `RESTART_REQUIRED_FIELDS = {"STACKCT_EMAIL", "STACKCT_PASSWORD", "ANTHROPIC_API_KEY"}`

2. **`get_settings() -> Dict[str, str]`:**
   - Use `dotenv_values(str(ENV_PATH))` to read current values
   - For each key in `ALLOWED_FIELDS`:
     - If in `SENSITIVE_FIELDS` and has value: return `"••••••••"` for password, `"{first7}...{last4}"` for API key
     - Add `{KEY}_set: bool` for sensitive fields to indicate if configured
     - Return actual value for non-sensitive fields
   - Return dict of all settings

3. **`validate_settings(data: Dict) -> Tuple[bool, List[str]]`:**
   - Reject keys not in `ALLOWED_FIELDS`
   - Validate email format (contains @) for `STACKCT_EMAIL`
   - Validate API key prefix (starts with `sk-ant-`) for `ANTHROPIC_API_KEY`
   - Validate `CLAUDE_MODEL` and `CLAUDE_MODEL_SCHEDULES` against `VALID_MODELS`
   - Validate numeric fields are positive integers
   - Return `(is_valid, list_of_errors)`

4. **`update_settings(data: Dict) -> Tuple[bool, str, bool]`:**
   - Call `validate_settings()` first
   - For each valid key-value pair:
     - Skip if key in `SENSITIVE_FIELDS` and value is empty string
     - Use `set_key(str(ENV_PATH), key, str(value), quote_mode="auto")` to write
     - Track if any `RESTART_REQUIRED_FIELDS` were changed
   - Return `(success, message, restart_required)`
  </action>
  <verify>
`python3 -c "from settings import get_settings, update_settings; print(get_settings())"` shows redacted values; sensitive fields show `••••••••` or partial API key, not actual values
  </verify>
  <done>Settings module provides secure read/write operations</done>
</task>

Add route:
```python
@app.route("/api/settings", methods=["PUT"])
def api_update_settings():
    """Update settings - partial updates supported."""
    data = request.get_json() or {}
    
    if not data:
        return jsonify({"error": "No settings provided"}), 400
    
    success, message, restart_required = update_settings(data)
    
    if not success:
        return jsonify({"error": message}), 400
    
    return jsonify({
        "success": True,
        "message": message,
        "restart_required": restart_required,
        "settings": get_settings()
    })
```
