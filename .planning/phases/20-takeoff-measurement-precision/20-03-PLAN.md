   - Start with `measure_result.copy()`
   - For components: build `seen_names` set from measure components (lowercased)

     - If name not in seen_names → append to merged components
     - If name IS in seen_names AND count-pass has `confidence="high"` AND existing has `quantity=None` → update quantity
     - If name IS in seen_names AND count-pass has `confidence="high"` → prefer count-pass quantity (overwrite)
   - If schedule_result provided → set `merged["schedules"] = schedule_result.get("schedules", [])`
   - Return merged dict

Keep backward compatibility: calling `analyze_drawing(img, sheet)` without `pass_type` uses "measure" (same as before).
