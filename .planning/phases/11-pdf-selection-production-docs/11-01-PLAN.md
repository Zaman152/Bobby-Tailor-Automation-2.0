### 1. Backend: PDF Metadata Extraction

**File:** `pdf_analyzer.py`

Add helper function to extract PDF metadata without full analysis:

```python
def get_pdf_metadata(pdf_path: str) -> dict:
    """Extract page count, file size, and sheet names without rendering."""
    import os
    doc = fitz.open(pdf_path)
    pages = []
    for i in range(len(doc)):
        sheet_name = _sheet_name(pdf_path, i)
        pages.append({"page_num": i + 1, "sheet_name": sheet_name})
    page_count = len(doc)
    doc.close()
    
    return {
        "page_count": page_count,
        "file_size_bytes": os.path.getsize(pdf_path),
        "pages": pages
    }
```

### 2. Backend: Upload Endpoint

**File:** `app.py`

Add new endpoint that uploads PDF and returns metadata (no analysis started):

```python
@app.route("/api/pdf/upload", methods=["POST"])
def upload_pdf():
    """Upload a PDF and return metadata for page selection."""
    if "file" not in request.files:
        return jsonify({"error": "No file uploaded"}), 400
    
    f = request.files["file"]
    if not f.filename.endswith(".pdf"):
        return jsonify({"error": "Only PDF files accepted"}), 400
    
    upload_id = str(uuid.uuid4())[:8]
    save_path = os.path.join("uploads", f"{upload_id}_{f.filename}")
    f.save(save_path)
    
    from pdf_analyzer import get_pdf_metadata
    meta = get_pdf_metadata(save_path)
    
    return jsonify({
        "upload_id": upload_id,
        "filename": f.filename,
        "save_path": save_path,  # internal use for run endpoint
        "page_count": meta["page_count"],
        "file_size_bytes": meta["file_size_bytes"],
        "pages": meta["pages"]
    })
```

Add in-memory upload tracker (similar to jobs dict):

### 4. Frontend: Modified Run Button

Change "Analyze PDF" button to send `upload_id` and `selected_pages`:

```javascript
async function runPDF() {
  const mode = document.querySelector('input[name="pageMode"]:checked').value;
  let selectedPages = null;
  
  if (mode === 'select') {
    selectedPages = [];
    document.querySelectorAll('#pageList input:checked').forEach(cb => {
      selectedPages.push(parseInt(cb.value));
    });
    if (selectedPages.length === 0) {
      alert('Please select at least one page');
      return;
    }
  }
  
  const res = await fetch('/api/pdf/run', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
      upload_id: currentUpload.upload_id,
      project_name: document.getElementById('pdfProjectName').value,
      selected_pages: selectedPages
    })
  });
  // ... poll job as before
}
```
