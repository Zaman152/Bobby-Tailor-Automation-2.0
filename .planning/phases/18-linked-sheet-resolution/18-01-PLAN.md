  <name>Task 1: Create linked_sheets.py with matcher + collector</name>
  <files>linked_sheets.py</files>
  <action>
Create `linked_sheets.py` at project root with two public functions:

**`match_ref_to_page(ref_sheet: str, catalog: list[dict]) -> Optional[dict]`**
- `catalog` is the list returned by `stackct_store.get_plans(stackct_id, folder_id)`,
  each dict has keys: `page_id`, `sheet_name`, `sheet_type`, `folder_id`.
- Normalize both ref_sheet and each sheet_name: uppercase, strip whitespace,
  replace `/` and `.` with `-`.
- For each catalog entry, compute a score (int):
  - 3 points: sheet_name starts with normalized ref (e.g. "C-4 - CIVIL..." matches "C-4")
    Strategy: split sheet_name on first " - " to get prefix, compare to normalized ref.
  - 2 points: normalized ref is a substring of normalized sheet_name
  - 1 point: normalized sheet_name ends with normalized ref
- Keep only entries with score > 0. If none, return None and log a WARNING with near-miss
  candidates (up to 3 closest sheet_names by character distance using difflib.get_close_matches).
- If multiple match: pick highest score; if tied, pick shortest sheet_name.
- Log match at DEBUG level including score and sheet_name matched.
- Return the matching catalog dict (with page_id, sheet_name, folder_id).
- Do NOT call get_plans() internally — caller passes catalog.

**`collect_unresolved_refs(all_extracted: list[dict], already_in_run: set[int]) -> list[dict]`**
- Walk every extraction dict in all_extracted.
- Collect ref_sheets from two sources:
  1. `extracted["cross_references"][].ref_sheet` (string field)
  2. `extracted["civil_structures"][].detail_ref_sheet` (string field)
- For each ref found, build a record:
  `{"ref_sheet": str, "from_sheet": extracted["_source_sheet"], "source": "cross_ref"|"civil_structure"}`
- Deduplicate by `ref_sheet` (case-insensitive): keep first occurrence.
- Exclude any ref_sheet that is empty/None.
- Returns list of ref records (not yet matched to page_ids — matching is separate).
- Caller will then match each to catalog and filter against `already_in_run`.
  `already_in_run` is a set of page_ids already in the manifest.

Module-level logger: `logger = logging.getLogger(__name__)`.
Type hints on all functions. Google-style docstrings.
