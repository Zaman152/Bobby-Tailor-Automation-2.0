
### T5: Update component-specific styles

**File:** `templates/index.html`

Components to update with tokens:

1. **Cards:** Background, border
2. **Buttons:** Primary gradient → solid primary, border radius
3. **Form inputs:** Border, focus state
4. **Progress bars:** Background, fill gradient → solid
5. **Status badges:** Use semantic colors (success, warning, danger)
6. **Log box:** Background, text color

Example updates:
```css
.card {
  background: var(--bg-surface);
  border: 1px solid var(--border-subtle);
  border-radius: var(--radius-lg);
}

.run-btn {
  background: var(--accent-primary);
  border-radius: var(--radius-md);
}

.run-btn:hover {
  background: var(--accent-secondary);
}
```

### T6: Adjust button gradient to match industrial style

**File:** `templates/index.html`

Current: `linear-gradient(135deg, #2563eb, #7c3aed)` (blue-purple)

Options:
- **Solid primary:** `var(--accent-primary)` — cleaner industrial look
- **Subtle gradient:** `linear-gradient(135deg, var(--accent-primary), var(--accent-secondary))`

Recommend: Solid primary with hover state transitioning to secondary.

```css
.run-btn {
  background: var(--accent-primary);
  transition: background 0.15s;
}

.run-btn:hover {
  background: var(--accent-secondary);
}
```

**Estimate:** Trivial
