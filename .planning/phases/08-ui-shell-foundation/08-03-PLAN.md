If custom config exists, ensure it points to `static/`.

**Estimate:** Trivial (verification only)

**Estimate:** Trivial

### T7: Update function references for extracted JS

**File:** `static/app.js`

If any `onclick` handlers in HTML reference global functions, ensure functions remain globally accessible:

**Current HTML:**
```html

```

**JS must expose:**
```javascript
// Inside DOMContentLoaded
window.runStackCT = runStackCT;
window.runPDF = runPDF;
```


**Recommend:** Keep `onclick` for now to minimize changes; expose functions globally.

**Estimate:** Small
